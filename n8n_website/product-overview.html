Product Overview

What is n8n?

n8n is an open-source, extendable workflow automation tool that follows a "fair-code" model.

Connect anything to everything

It provides a node-based, low-code interface to connect different apps and APIs, making it highly versatile for automating tasks.

Users can:

Build workflows by dragging nodes (representing services or logic) on a canvas

Inject custom code (JavaScript, Python, etc.) within workflows for ultimate flexibility

Self-host with custom functions or integrations, or use n8n's cloud service

Problem & Goal

The Problem

n8n was created out of frustration with existing workflow tools that "either lacked flexibility or required deep technical expertise."

Users needed a solution that could handle complex integrations without the limitations of closed-source tools or the complexity of custom coding.

The Goal

Build a solution "powerful enough for developers yet accessible to anyone willing to learn."

n8n aims to:

  • Give technical teams the power of custom code automation without reinventing the wheel
  • Enable different services to talk to each other in automated pipelines
  • Bridge the gap between simple no-code tools and full custom scripting
  • Give technical people "the powers of a 10x developer" through automation

Key Features

500+ Built-in Integrations

Pre-built nodes for popular services and APIs, with a visual editor to construct workflows

Self-hosting Option

Run on your own server for data control and unlimited usage, or use n8n's cloud service

In-workflow Coding

Add custom JavaScript, Python, or other code within workflows for ultimate flexibility

Extensibility

Extend the platform with community-contributed or custom nodes

Multiple Triggers

Start workflows via webhooks, timers, or triggers from integrated apps

Complex Logic

Handle branching logic, loops, and data transformations in a flowchart style
